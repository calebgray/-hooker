# $hooker

Even more vanilla than most prefer.


## Usage

When you tell $hooker to `$hook(this, 'key', function(trigger){})`; yer regeristin' yer `$hook`...

... and when you tell `$hooker` to `$hook(this)`; yer tellin' 'em to fire yer `$hooks`.
